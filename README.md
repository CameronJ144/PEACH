# Georgia P.E.A.C.H. Campaign

A campaign website with a hidden QR‑code form and JSON‑based storage.

## Features

- Responsive landing page with hero, three columns, FAQ, and footer.
- Hidden form accessible via `/qr-form` or a modal on the main page.
- Backend stores submissions in `submissions.json` with points logic.
